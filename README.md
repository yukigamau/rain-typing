## 简介

雨天跟打器是一款免费开源的便携打字练习软件，该软件基于[添雨跟打器旧版本](https://github.com/taliove/tygdq)项目的修改构建项目[雨天跟打器](https://github.com/LightAPIs/ytgdq)上创建的，主要是跟据本人的使用习惯进行了微调。
